# Codex Instructions

- Treat any manual edits in this repository as user-owned work.
- Do not undo, overwrite, revert, or discard changes made manually by the user unless the user explicitly asks for that specific revert.
- Before editing a file that already has uncommitted changes, read the surrounding code and preserve the user's changes while making the requested update.
- If user edits conflict with the requested task, explain the conflict and ask how to proceed instead of silently replacing their work.
